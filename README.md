# Shiny GA feature selection


### Install

Example with new environment named `shiny`

```
conda create -n shiny python=3.9 -y

conda activate shiny

conda install pip -y

pip install -r requirements.txt
 ```

You can then run shiny lik:

`shiny run --reload app.py`
